Anthropic's protocol for extending Claude's capabilities via external tools, resources, and prompts. Central integration layer across the knowledge system.

## Key Facts

- Used in: qmd (vault search), memory MCP (entity graph), GitNexus (code intelligence), Pencil (design)
- Configured in `~/.claude.json`
- Auto-approved tools in `settings.json`
- Enables Claude Code to search Obsidian, manage memory, query code graphs
