Claude Code plugin for multi-agent orchestration. 19 agents with smart model routing (Haiku for search, Sonnet for execution, Opus for architecture). v4.9.3.

## Key Facts

- Install: `npm i -g oh-my-claude-sisyphus@latest && omc install && omc setup`
- Magic keywords: `autopilot:`, `ralph:`, `ulw`, `deep-interview`, `deepsearch`, `ultrathink:`
- Agents: explore, analyst, planner, architect, critic, code-simplifier, debugger, executor, verifier, designer, test-engineer, scientist, tracer, qa-tester, git-master, document-specialist, writer
- Model routing: Haiku (90% of Sonnet, 3x savings) for lightweight tasks
