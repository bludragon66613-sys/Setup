React meta-framework used across the majority of Rohan's projects. App Router pattern, deployed on Vercel.

## Key Facts

- Version: 16.x across projects
- Used in: Aeon Dashboard, Paperclip, Munshi, NERV dashboard
- Pattern: App Router (not Pages Router)
- Turbopack: disabled on Windows (bug), use `--webpack` flag
- Deployment: Vercel (Fluid Compute)
