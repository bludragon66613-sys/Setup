Knowledge graph and vault system. Layer 2 of the 3-layer memory architecture. The "IDE" in Karpathy's analogy — Claude writes the wiki, Obsidian renders it.

## Key Facts

- Vault: `~/OneDrive/Documents/Agentic knowledge/`
- Search: qmd MCP (BM25 + vector embeddings)
- Plugins: Dataview, Web Clipper, Marp (slides), arscontexta
- Graph view for visualizing wiki structure
- MindMap.md auto-rebuilt on session start/stop
