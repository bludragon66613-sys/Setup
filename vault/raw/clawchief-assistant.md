## Summary
Ryan installed clawchief (github.com/snarktank/clawchief) as an operating layer on OpenClaw to create an executive assistant + chief of staff.

## What It Does
- Schedule meetings, check inbox every 15 min, follow up on emails
- Watch calendar, flag conflicts, prep task lists before waking up
- Keep task list clean across sessions
- Update CRM, research suppliers, send short high-signal updates
- Runs from durable context in files, memory, Gmail, Calendar, Sheets

## Golden Rule
> "The best version reflects your actual world. Generic assistants are generic because they are under-configured."

## What We Implemented
- HEARTBEAT.md upgraded to proactive checks
- tasks/current.md created as single source
- 2 AM task prep cron job set up
- Clawchief skills installed (executive-assistant, daily-task-manager, daily-task-prep, business-development)
