# NERV / SIGNAL / Paperclip / OpenClaw

Rohan's AI agent consultancy + product stack. Pre-launch, 1 client acquired via personal connection.

## Brand architecture (as of 2026-03-26)

```
NERV (parent consultancy)
  ├── SIGNAL (product/platform sub-brand — the autonomous agent engine)
  ├── Paperclip (open-source credibility signal)
  └── OpenClaw (AI gateway infrastructure)
```

**NERV signs contracts and sends invoices.** Not SIGNAL. This overrides earlier recommendations.

## Each brand's role

| Brand | Role | Audience |
|-------|------|----------|
| **NERV** | Parent consultancy. Contracts, invoices, client relationships, strategic positioning. | Enterprise + SME decision-makers |
| **SIGNAL** | Productized autonomous agent engine. The thing NERV sells. | Same as NERV, framed as outcomes not tech |
| **Paperclip** | Open-source agent orchestration platform. Credibility signal, not a revenue line. | Developers, potential hires, press |
| **OpenClaw** | Local AI gateway infrastructure. Powers Telegram bot + internal tooling. | Operational, not customer-facing |

## Positioning rules

1. **Never sell "agents."** Sell outcomes. Example: "5 AI employees, INR 5K/month" — not "multi-agent LLM orchestration with tool use."
2. **India first, global second.** India is proximity + underserved. Global comes after case studies.
3. **Service business first. SaaS later.** Phase 2 move.
4. **WhatsApp-first for India MSME delivery.** Moat vs Zoho/Freshworks.

## Do not

- Position SIGNAL as the parent brand (NERV is parent)
- Mention "agents" or "LLMs" to non-technical SME buyers — lead with outcomes
- Lead with SaaS — lead with service
- Under-price Global tiers below $149 or above $899
- Confuse Paperclip (OSS, credibility) with SIGNAL (productized, revenue)
